1. Información que recopilamos

EHS Guardian accede a datos de sensores como luxómetro y acelerómetro, exclusivamente para su funcionamiento. No recopilamos ni almacenamos datos personales del usuario.

2. Uso de permisos

La aplicación solicita permisos para acceder a sensores del dispositivo (luz y movimiento). Estos permisos se utilizan únicamente para mostrar mediciones en tiempo real y no se guardan ni comparten con terceros.

3. Seguridad de los datos

EHS Guardian no almacena ni comparte datos del usuario. Todas las mediciones son procesadas localmente en el dispositivo.

4. Modificaciones a esta política

Podemos actualizar esta política de privacidad. Cualquier cambio se reflejará en la app y en nuestra página oficial.
